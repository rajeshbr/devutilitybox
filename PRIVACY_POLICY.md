## 3. What Personal Data We Collect

### 3.1 Data You Provide Directly

**None collected by default.** All data processing occurs entirely on your device using browser localStorage. We do not transmit your data to our servers.

- **JSON Editor Input:** Stored only in your browser's localStorage, not sent to any server
- **Code Editor Content:** Stored only in your browser's localStorage
- **Base64 Input/Output:** Processed and stored locally only
- **List Comparator Data:** Stored locally in your browser
- **SQL Formatter Input:** Stored locally in your browser
- **Timezone Converter Settings:** Stored locally in your browser
- **JWT Token Input:** Decoded locally in your browser, not transmitted

### 3.2 Technical Data Collected

If you use our service and we implement analytics (currently not enabled), we may collect:
- IP address (anonymized)
- Browser type and version
- Operating system
- Page visit duration
- Referrer information
- Approximate geographic location (country level only)

**Current Status:** No analytics data is currently collected.

## 6. Data Storage & Security

### 6.1 Local Storage
- All user input data is stored in your browser's localStorage
- No data is transmitted to our servers
- Data persists between sessions on your device
- Clearing browser cache/localStorage will erase all data

### 6.2 Security Measures

We implement the following security measures:

- **HTTPS/TLS Encryption:** All communications use encrypted connections
- **No Server-Side Storage:** Data is processed client-side only
- **Regular Security Audits:** Code is reviewed for vulnerabilities
- **Dependency Updates:** Regular security patches applied
- **No Third-Party Data Sharing:** Your data is never shared with external services

## 7. Data Retention

- **localStorage Data:** Retained until you manually delete it
- **Browser Cookies:** Essential cookies retained for session; optional cookies as per your consent
- **No Server Records:** We maintain no server-side logs of your input data

## 8. Your Rights Under GDPR

You have the following rights regarding your personal data:

### 8.1 Right to Access (Article 15)
Request a copy of all personal data we hold about you.

### 8.2 Right to Rectification (Article 16)
Request correction of inaccurate personal data. Since we store data locally, you can directly edit it in the application.

### 8.3 Right to Erasure (Article 17)
Request deletion of your personal data. You can delete localStorage data directly by:
- Clearing your browser cache
- Using the "Clear" button in each tool
- Adjusting browser settings

### 8.4 Right to Restrict Processing (Article 18)
Request restriction of data processing. You can do this by:
- Disabling non-essential cookies
- Clearing localStorage data

### 8.5 Right to Data Portability (Article 20)
Request your data in a structured, commonly used format. You can export localStorage data using browser developer tools.

### 8.6 Right to Object (Article 21)
Object to processing of your data. Contact us at [contact@devutilitybox.com]

### 8.7 Right to Lodge a Complaint
Lodge a complaint with your local Data Protection Authority.
